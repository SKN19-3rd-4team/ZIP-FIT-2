5. 정렬 문제 오류 파악
예) 필터 설정 : 전체 기관 | 행복 주택 | 전체 상태
페이지 2개로 출력됨
첫 페이지 - > 공고중, 접수중, 공고예정, 마감 정렬 규칙 올바르게 되어있음.
두번째 페이지 -> 공고예정, 마감 으로 되어있음.
-> 즉 두번째 페이지에서도 정렬 규칙을 따르고 있는 것으로 파악된다.
다만, 두번째 페이지의 '이천 행복주택 입주자 모집공고'는 공고예정이니, 첫번째 페이지에 넘어가야하지 않을까 싶다.
왜냐하면 첫번째 페이지에 지금 공고중, 접수중, 공고예정, 마감 -> 마감에 대한 공고카드도 있다.
사용자는 첫페이지에서 마감 공고를 바라보고 있는 것, 두번째 페이지 넘어가면 '어? 공고 예정이 여기 또있네?'
라는 형태가 됨.

-> 문제 해결 제시바람.
