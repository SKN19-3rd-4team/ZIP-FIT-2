**원인 분석**:

- ID 불일치 및 함수 호출 방식 불일치

**해결 방안**:
- 모든 페이지에서 동일한 함수 사용
- ID 통일 (`chatHistoryList`)

## 🔍 발견된 문제 상세

### 문제 1: API 호출 방식 불일치
- `main.html`: `API.getChatHistories()` 사용
- `chat.html`, `list.html`: `callApi()` 사용
- **해결**: 모든 페이지에서 `callApi()` 사용하도록 통일

### 문제 2: 채팅 히스토리 컨테이너 ID 불일치
- `main.html`: `chatHistoryContainer`
- `chat.html`, `list.html`: `chatHistoryList`
- **해결**: `chatHistoryList`로 통일

### 문제 3: 채팅 폼 제출 시 새로고침
- **원인**: 폼 제출 이벤트에서 `e.preventDefault()` 호출 안됨
- **해결**: 이벤트 리스너에 `e.preventDefault()` 추가 확인

### 문제 4: 색상 문제
