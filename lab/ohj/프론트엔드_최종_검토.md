## ✅ 8. 발견된 잠재적 이슈

### 8.1 URL 네임스페이스
**현재 상태**:
```python
# web/urls.py
urlpatterns = [
    path('', views.landing_view, name='landing'),
    ...
]
```

**템플릿에서 사용**:
```html
{% url 'web:main' %}
```

⚠️ **이슈**: `web/urls.py`에 네임스페이스가 없음

**해결 방법**:
```python
# config/urls.py
urlpatterns = [
    path('', include(('web.urls', 'web'), namespace='web')),  # 네임스페이스 추가
    ...
]
```

또는 템플릿에서:
```html
{% url 'main' %}  # 네임스페이스 없이 사용
```

### 8.2 index.html 리다이렉트
**figma14**:
- `index.html`이 `landing.html`로 리다이렉트

**figma_django**:
- `/`가 이미 `landing_view`로 연결됨
- 별도 `index.html` 불필요

✅ **정상**: Django URL 구조로 자동 해결
